# AI Compiler 🚀

This is a Laravel-based AI-powered multi-language code compiler.

## 🔧 Features
- Multi-language code execution
- AI-based suggestions
- Clean UI
- Fast processing

## 🛠 Tech Stack
- Laravel (PHP)
- MySQL
- Docker (optional)

## 🚀 Setup Instructions

```bash
git clone https://github.com/Kriyabishnoi/aiCompiler.git
cd aiCompiler
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
